# Outgoing Webhook Payloads

Outgoing webhooks supports 2 versions of events and the payload format differs between these versions.

## v1

The v1 supports only the below 4 events and their payloads are linked below.

* [Incident triggered](../../payload/v1/incident-triggered/)
* [Incident Reassigned](../../payload/v1/incident-reassigned/)
* [Incident Acknowledged](../../payload/v1/incident-acknowledged/)
* [Incident Resolved](../../payload/v1/incident-resolved/)

## v2

The v2 supports much more exhaustive set of Incident events & actions and their payloads are linked below.

* [Incident triggered](../../payload/v2/incident-triggered/)
* [Incident Reassigned](../../payload/v2/incident-reassigned/)
* [Incident Acknowledged](../../payload/v2/incident-acknowledged/)
* [Incident Resolved](../../payload/v2/incident-resolved/)
* [SLO Violating Incident](../../payload/v2/slo-violating-incident/)
* [SLO violation false positive](../../payload/v2/slo-violation-false-positive/)
* [SLO violation unmark false positive](../../payload/v2/slo-violation-unmark-false-positive/)
* [SLO Violating Incident Error budget Updated](../../payload/v2/slo-violating-incident-update-error-budget/)
* [Incident Notes Created](../../payload/v2/incident-notes-created/)
* [Incident Notes Updated](../../payload/v2/incident-notes-updated/)
* [Incident Notes Deleted](../../payload/v2/incident-notes-deleted/)
* [Incident Note Starred](../../payload/v2/incident-note-starred/)
* [Incident Note Unstarred](../../payload/v2/incident-note-unstarred/)
* [Incident Tags Updated](../../payload/v2/incident-tags-updated/)
* [Incident Task Created](../../payload/v2/incident-task-created/)
