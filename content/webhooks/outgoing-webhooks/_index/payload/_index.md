# Outgoing Webhook Payloads

Outgoing webhooks supports 2 versions of events and the payload format differs between these versions.

## v1

The v1 supports only the below 4 events and their payloads are linked below.

* [Incident triggered](v1/incident-triggered.md)
* [Incident Reassigned](v1/incident-reassigned.md)
* [Incident Acknowledged](v1/incident-acknowledged.md)
* [Incident Resolved](v1/incident-resolved.md)

## v2

The v2 supports much more exhaustive set of Incident events & actions and their payloads are linked below.

* [Incident triggered](v2/incident-triggered.md)
* [Incident Reassigned](v2/incident-reassigned.md)
* [Incident Acknowledged](v2/incident-acknowledged.md)
* [Incident Resolved](v2/incident-resolved.md)
* [SLO Violating Incident](v2/slo-violating-incident.md)
* [SLO violation false positive](v2/slo-violation-false-positive.md)
* [SLO violation unmark false positive](v2/slo-violation-unmark-false-positive.md)
* [SLO Violating Incident Error budget Updated](v2/slo-violating-incident-update-error-budget.md)
* [Incident Notes Created](v2/incident-notes-created.md)
* [Incident Notes Updated](v2/incident-notes-updated.md)
* [Incident Notes Deleted](v2/incident-notes-deleted.md)
* [Incident Note Starred](v2/incident-note-starred.md)
* [Incident Note Unstarred](v2/incident-note-unstarred.md)
* [Incident Tags Updated](v2/incident-tags-updated.md)
* [Incident Task Created](v2/incident-task-created.md)
* [Incident Task Updated](v2/incident-task-updated.md)
* [Incident Task Deleted](v2/incident-task-deleted.md)
* [Incident Task Completed](v2/incident-task-completed.md)
* [Incident Task Uncompleted](v2/incident-task-uncompleted.md)
* [Incident Communication Channel Created](v2/incident-communication-channel-created.md)
* [Incident Communication Channel Updated](v2/incident-communication-channel-updated.md)
* [Incident Communication Channel Deleted](v2/incident-communication-channel-deleted.md)
* [Statuspage Updated](v2/statuspage-updated.md)
